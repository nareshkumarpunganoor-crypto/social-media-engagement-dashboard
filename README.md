# Social Media Engagement Dashboard

## Project Overview
This project analyzes social media engagement data using Python and Power BI.

The dashboard tracks:
- Likes
- Comments
- Shares
- Views
- Engagement Rate

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Power BI

## Dataset
The dataset contains 10,000 social media posts with engagement metrics.

## Features
- Data cleaning
- Exploratory Data Analysis
- KPI calculations
- Interactive Power BI dashboard
- Platform-wise analysis
- Post-type performance analysis

## Dashboard Insights
- Video posts generate higher engagement
- Instagram receives highest interactions
- Views positively correlate with likes
